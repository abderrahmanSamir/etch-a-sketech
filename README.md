# etch-a-sketech

À la fin de cet exercice, vous devez avoir consolidé les compétences fondamentales suivantes en développement web :

1. Manipulation du DOM avec JavaScript
   Créer dynamiquement des éléments HTML (div) via JavaScript.
   Ajouter, supprimer ou modifier des éléments et leurs attributs.
   Comprendre la structure hiérarchique du DOM (ex. : insertion dans un conteneur).
2. Utilisation des événements
   Mettre en place des écouteurs d’événements (event listeners) pour détecter le passage de la souris (mouseover, mouseenter, mouseleave).
   Gérer l’interaction utilisateur en temps réel.
3. Boucles et logique conditionnelle
   Utiliser des boucles (for, forEach) pour générer une grille de taille variable (ex. : 16x16 ou 64x64).
   Appliquer une logique conditionnelle pour limiter l’entrée utilisateur (≤ 100 cases).
4. Interaction avec l’utilisateur
   Utiliser prompt() pour demander une saisie.
   Gérer le bouton avec onclick ou addEventListener.
5. Styling dynamique avec CSS et JavaScript
   Appliquer des styles via JavaScript (ex. : changement de backgroundColor).
   Comprendre l’effet des bordures, marges et box-sizing sur la mise en page.
   Utiliser Flexbox pour disposer les éléments en grille (sans recourir à CSS Grid).
6. Gestion de l’état et effets visuels (crédit supplémentaire)
   Générer des couleurs aléatoires en RGB.
   Implémenter un assombrissement progressif via la propriété opacity ou en ajustant les valeurs de couleur.
7. Bonnes pratiques de développement
   Séparer logique (JavaScript), structure (HTML) et présentation (CSS).
   Utiliser les outils de développement (console, éléments, erreurs) pour déboguer.
   Structurer le code de manière modulaire (ex. : fonctions pour créer la grille, réinitialiser, etc.).
